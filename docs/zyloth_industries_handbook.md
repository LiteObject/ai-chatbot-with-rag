## Important Policies

### Code ZY-47: Remote Work Policy
Employees may work remotely up to 3 days per week. Remote work requests exceeding this must be approved by direct manager and HR using Form RW-2024-B. The maximum remote work radius is 500 kilometers from the nearest office (due to quantum sync requirements).

### Code ZY-112: Invention Disclosure
All inventions created during work hours or using company resources must be disclosed within 14 business days using the Zyloth Innovation Portal. Employees receive 15% royalty on any commercialized inventions.

### Code ZY-89: Dress Code
Business casual is required Monday-Thursday. Fridays are designated "Expression Days" with no dress code restrictions (excluding safety-critical areas). Lab coats in research areas must be regulation purple (#7B2D8E).
